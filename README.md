📚 Study Tracker App

PythonとStreamlitで作成した勉強時間管理アプリです。

主な機能
勉強記録の追加
科目別勉強時間の集計
今日の勉強時間表示
今週の勉強時間表示
目標達成率表示
連続勉強日数（ストリーク）
科目別ランキング
棒グラフ表示
円グラフ表示
週間推移グラフ
勉強カレンダー表示
使用技術
Python
Streamlit
Matplotlib
起動方法
pip install streamlit matplotlib
streamlit run app.py
